# Docs Archive

This archive keeps older documentation that is still useful for history and debugging, but should not be treated as current guidance.

## Folders

- `notes/`
  Temporary notes, patch summaries, and step-by-step migration snapshots.

- `stages/`
  Historical stage documents from earlier Elira migration and feature rollout work.

## Use it when

- you need to understand how an old migration step was done;
- you are tracing a legacy design decision;
- you need historical context for cleanup work.

For current startup, runtime, and stabilization guidance, use:

- `README_Elira_AI.md`
