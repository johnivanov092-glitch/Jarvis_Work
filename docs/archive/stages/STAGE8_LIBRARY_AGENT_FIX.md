# Stage 8A — Library Visibility Fix

Этот патч делает две вещи:
1. Добавляет /api/library/context
2. Заставляет агента читать до 3 активных текстовых файлов, а не только видеть их имена

Что это исправляет:
- агент теперь видит не просто список активных файлов
- в LLM попадает реальный текст активных файлов
- в timeline видно, какие файлы были реально прочитаны
